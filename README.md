# 🌀 Yumi UserBot

> **Yumi** — это мощный, модульный и быстрый BotNet на Python, построенный на базе Telethon. Создан для автоматизации,  фана.

---

## 🚀 Возможности

*   **⚡️ High Performance** — оптимизированный код, минимальный отклик.
*   **📂 Modular System** — легкая установка и создание новых модулей через шаблоны.
*   **🔍 WhatIs Module** — детальный пробив пользователей (ID, дата регистрации, статус скама).
*   **👻 Ghost Mode** — режим невидимости и логирование удаленных сообщений.
*   **🎮 Fun & Animations** — динамические анимации прямо в чате (`.police`, `.love`).
*   **🖼 Rich Ping** — красивый пинг с отображением аптайма и вложенным изображением.
